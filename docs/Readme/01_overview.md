# 📱 01 — アプリ概要

## 概要

Whispinは、ユーザーが**匿名でリアルタイムチャット**を楽しめるFlutterアプリケーションです。  
ルームを作成し、他のユーザーと**時間制限付き**のチャットセッションを行えます。

---

## 主要機能

### チャット
| 機能 | 説明 |
|---|---|
| 匿名チャット | ニックネームを使用した匿名チャット |
| ルームベース | パブリック / プライベートルームの作成・参加 |
| 時間制限 | 10分間の制限時間付き（延長可能） |
| 評価システム | チャット後の相手評価（👍 / 👎） |

### ソーシャル
| 機能 | 説明 |
|---|---|
| フレンド機能 | ユーザー間のフレンド申請・管理 |
| 招待機能 | フレンドをプライベートルームに招待 |
| ブロック機能 | 特定ユーザーのブロック |
| 通知機能 | リアルタイム通知（フレンドリクエスト、招待） |

### その他
| 機能 | 説明 |
|---|---|
| プレミアムプラン | 延長回数無制限などの特典 |
| 管理者機能 | ユーザー管理、ログ閲覧、お問い合わせ対応 |
| レスポンシブデザイン | モバイル・タブレット・デスクトップ対応 |
| ダークモード | テーマ切り替え対応 |
